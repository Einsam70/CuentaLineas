# 1. Función

Utilidad muy simple, únicamente para contar lineas de texto/código dentro de cualquier tipo de archivo.

Por defecto seleccionará archivos *.txt y *.csv, pero puede contar las líneas en documentos de cualquier tipo.

# 2. Como utilizar

El único archivo necesario es "index.html". Lo puedes descargar y ubicar en cualquier lugar de tu equipo, unidad USB, disco externo, etc. No necesita ninguna libreria adicional.
